# The-Nim-Game
Intro to AI course final project:
After this assignment, the student should be able to:
1.	Design and implement a tree structure that encodes the search space of a given search problem.
2.	Implement and use a standard tree search techniques.
3.	Demonstrate how to use search techniques concepts to build a winning strategy.
4.	Describe the advantages and the limitations of a search technique.
Problem
A game software company approached you asking to design for them a software for the Nim game.  The requirement as follows 
1.	The game is to be played by the user versus the machine 
2.	It should be interactive
3.	The objects (or their symbols) should be displayed 
4.	The software should give to the user the option whether or not he wants to make the first move.
Also, as the software is meant to be used by kids, the game should look simple, and not taking so too much time. For this purpose, the number of pile is set to one, and the number of objects should be set to 15,  and  each player can take only one, two, or three objects from the pile.

The solution must contain a search technique must be used for the creation of the AI opponent
